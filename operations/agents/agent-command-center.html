Agent Council Command Center

Start here to run the lAIdies agent organization: weekly command center first, whole-site review next, Fun & Games as the stress test, then CEO Top 5 actions.

Run Order

  1. Open the Weekly Command Center.
    Check Project Radar first: recent changes, launch blockers, Member Pass, Clubhouse QA, canon intake, and handover health.
  2. Open the visual org map.
    Confirm which senior agents and section owners are responsible for the review.
  3. Run the whole-site review.
    Judge cohesion, hierarchy, release state, learning path, polish, governance, and reputation before isolated fixes.
  4. Use Fun & Games as the stress test.
    Check for chaotic layout, stale open panels, blank space, fake interactions, fortune-teller reveal behavior, and brand/reputation risk.
  5. Score agents against a target of 5.
    Each agent explains why it did or did not earn a 5 and names the top-score action.
  6. Move blockers to CEO Top 5.
    No blocker stays buried inside a section note.
