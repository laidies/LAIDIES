# Article-to-Quiz Calibration Template

Purpose: verify that a quiz can be answered by a reader who has read only the relevant article/source material. This prevents quizzes from rewarding outside knowledge, punishing reasonable interpretations, or asking questions the article did not actually teach.

## Rules

- Read only the source material listed above.
- Do not use outside knowledge, author intent, or the answer key while taking the first pass.
- Mark confidence for each answer before checking the key.
- After checking the key, trace each correct answer back to the article/source material.
- Review the post-submit explanation and reread link for every question.
