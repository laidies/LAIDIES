## Signals To Check

- broken links
- console errors
- mobile layout defects
- quiz/card/game starts or completions
- newsletter clicks
- feedback submissions
- community comments
- stale issue state
- unresolved launch blockers
- user confusion or repeated questions

## Status Options

- no issue
- watching
- needs fix
- fixed
- escalated
- parked
