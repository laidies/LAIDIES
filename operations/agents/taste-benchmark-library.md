# Taste Benchmark Library

This library calibrates lAIdies taste. Use it when reviewing content, writing new sections, or deciding whether an idea is funny, useful, both, neither, or reputationally risky.

## Core Taste Principle: Smart Women, Better Learning

lAIdies does not use playful references because women need technical ideas made silly. lAIdies uses playful references because smart professionals deserve learning that is accurate, useful, memorable, and enjoyable.

The reference should sharpen the concept, speed up recall, or make the next action feel doable. If the joke replaces the lesson, talks down to the reader, or makes the technical idea smaller than it is, it fails the brand.

## Gold Standard: Mme CLAI-O

Mme CLAI-O is protected as gold-standard lAIdies copy. The current fortunes are not a problem to optimize away; they are a benchmark for future writing.

Why it works:

- specific 90s/Y2K object language
- card / read / message / move structure
- practical next action
- sharp humor with warmth
- no generic tarot language
- no generic empowerment copy
- useful for a real professional moment

Examples from current copy:

### The Butterfly Clip

- Read: You are overthinking something that only needs one cute, decisive move.
- Message: The room does not need a ten-point thesis. It needs you to say the thing clearly.
- Move: Send the shorter version before your brain adds a subplot.

### The Caboodle

- Read: Your tools are everywhere, and your patience is under a butterfly clip somewhere.
- Message: The problem is not that you have no resources. The problem is that none of them have a home.
- Move: Put one prompt, one template, and one useful link in the same place today.

### The Magic 8 Ball

- Read: You already know the answer, but you would like a plastic orb to take liability.
- Message: Outlook good, but only if you stop outsourcing your judgment to anything round and mysterious.
- Move: Write the recommendation and the reason. Then send it to one reviewer.

## Funny And Useful

Use when a joke makes the practical action easier to remember.

Standard:

- specific object or situation
- one useful workplace insight
- a line someone might quote
- no punch-down energy

## Funny But Not Useful

Use as a warning category. This copy may get a laugh but does not help the reader learn, decide, try, or remember.

Common signs:

- reference does all the work
- no next action
- joke is too inside-baseball
- concept gets blurrier after the joke

## Useful But Too Beige

Use as a warning category. This copy is technically fine but could appear in any workplace newsletter.

Common signs:

- no Ali voice
- no image, analogy, or concrete example
- sounds like a generic productivity blog
- too many abstract nouns

## Too AI-Influencer

Use as a warning category. lAIdies should not sound like a hype account.

Common signs:

- "10x your life" energy
- vague claims about transformation
- overpromising automation
- treating AI as magic instead of a tool

## Too Sales-y

Use as a warning category. Money should feel like an extension of the lAIdies world, not an interruption.

Common signs:

- urgency tactics
- vague value claims
- sponsor/product plugs that interrupt learning
- paid offer before audience demand exists

## Technically Correct But Boring

Use when the fact is right but the reader experience is weak.

Fix by adding:

- a workplace example
- a stronger analogy
- a cleaner sequence
- a Try-On
- a glossary anchor

## Reputationally Risky / Needs Review

Use when copy may be funny or useful but could create avoidable risk.

Examples:

- jokes that undermine Ali's senior-professional credibility
- employer-specific implications
- named career opportunities
- legal, tax, HR, medical, financial, privacy, or policy guidance framed as advice
- claims about tools, models, studies, or laws without receipts
- community answers labeled correct before human review
