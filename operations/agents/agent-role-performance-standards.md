# Agent Role Performance Standards

Purpose: define what excellent performance means for every lAIdies Agent Council role. The CEO Agent uses this file to score agents. Every role always aims for a 5.

## Universal 5-Point CEO Standard

| Score | Meaning | CEO interpretation |
| --- | --- | --- |
| 5 | Best-in-class owner | Found the highest-leverage issue or opportunity, gave evidence, made a sharp tradeoff, protected lAIdies quality, and proposed an implementation-ready action. |
| 4 | Strong owner | Useful, specific, and mostly complete, but missed one integration, measurement, or execution detail. |
| 3 | Adequate reviewer | True but expected; identifies an issue without making the work meaningfully better. |
| 2 | Weak reviewer | Vague, generic, unsupported, too narrow, or misses an obvious responsibility. |
| 1 | Dud | Passes broken work, ignores its mandate, gives no evidence, or recommends something harmful, off-brand, or unmaintainable. |

## Universal Anti-Dud Rules

An agent cannot score above 3 if it:

- gives advice without evidence
- uses vague language instead of a concrete action
- ignores desktop/mobile behavior for website-facing work
- passes fake or invisible interactions
- ignores release state
- creates more work than Ali can maintain
- protects a pet idea over reader value
- flattens lAIdies voice into generic AI content
- misses a clear risk within its own mandate
