# Automation Docs

Automation plans for newsletter production, website updates, comment summaries, community moderation, and agents.

The weekly issue production system is in `docs/automation/weekly-production-system.md`. Use `.\scripts\run-weekly-production.ps1` before Ali review to rebuild generated outputs and produce the issue review packet.

The AI Chat Room Digest is a priority community automation. It should monitor Hyvor Talk comments, classify useful answers/tips/unanswered questions, flag risky claims for human review, and publish a navigable digest board. See `docs/community/ai-chat-room-digest-plan.md`.
