# Reference Closet Commerce And Watch Plan

Use this when turning the Reference Closet into a practical resource with product links, watch links, and issue tie-ins.

## Working Concept

The Reference Closet should be useful, not just nostalgic.

Each reference can have:

- the analogy
- the lAIdies lesson
- where to watch/read/listen
- where to buy a related object or modern equivalent

Possible feature names:

```text
Shop the Closet
```

```text
Watch the Closet
```

```text
Where to Find It
```

```text
Add to Cart, But Make It Canon
```

## Why It Works

This creates:

- a practical reader resource
- a fun reason to revisit the website
- an issue-by-issue affiliate/commerce path
- a way to make references legible for people who missed them
- a Soundtrack Saturday / Reference Closet social hook

## Important Guardrails

Do not publish shopping or streaming links without checking them that week.

Why:

- streaming availability changes
- prices change
- products disappear
- Amazon listings vary by country
- affiliate disclosure requirements matter

Default wording:

```text
Some links may be affiliate links. If you buy through them, lAIdies may earn a small commission at no extra cost to you.
```

Do not frame this as Amazon-branded.

Good:

```text
Shop the Closet
```

Avoid:

```text
Amazon picks from Ali
```
