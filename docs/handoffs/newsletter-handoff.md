# Newsletter Handoff

Read this before newsletter work.

## Purpose

Each newsletter episode is the source of truth for:

- website issue page
- quiz
- card pack
- glossary
- reference bank
- Instagram content
- playlist/House DJ
- community prompts

## Style

The writing should feel deeply human, funny, useful, and specific. It should not sound like AI wrote it.

Use:

- 90s/Y2K references
- corporate-woman reality
- practical explanations
- accurate analogies
- direct but warm advice

Avoid:

- generic AI hype
- vague “vibes” language
- condescension
- overexplaining basic points in a sterile way
