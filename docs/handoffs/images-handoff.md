# Images Handoff

Read this before image generation or asset work.

## Rule

Final assets used by the site must be saved inside `assets/`. Do not leave website-referenced images only in Codex generated-image folders.

## Asset Organization

New folders exist for future cleanup:

- `assets/brand/`
- `assets/episodes/`
- `assets/games/`
- `assets/community/`
- `assets/playlists/`
- `assets/people/`
- `assets/archive/`

Do not move existing assets yet unless all references are updated and tested.

## Style

Images should feel like the newsletter:

- 90s/Y2K
- pink/neon/teal where appropriate
- playful but polished
- useful visual signal, not random decoration
- no copied copyrighted characters/logos
