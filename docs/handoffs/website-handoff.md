# Website Handoff

Read this before doing website work.

## Current Site

The website is a static site with:

- `index.html`
- `styles.css`
- `script.js`
- `issues/`
- `community/`
- `assets/`

Local preview usually runs at:

`http://127.0.0.1:4173/index.html`

## Current UX Direction

The homepage should feel like a guided experience, not a full inventory of every idea.

Recommended order:

1. Hero
2. Current episode / start here
3. Learn
4. Community
5. Games
6. Resources
7. Origin story
8. Subscribe

## Important Rules

- Do not move live files unless links are updated and tested.
- Keep old quizzes available by issue.
- Keep old card packs available by issue.
- Games should reveal full modules only when the user chooses them.
- Use real/generated bitmap visuals for major features.
