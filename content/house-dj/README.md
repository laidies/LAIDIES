# House DJ

Store DJ JAIDY's weekly AI song links, prompts, descriptions, and episode pairings here.

Creative frame: DJ JAIDY is the lAIdies House DJ, the recurring club-scene anchor for the newsletter. Each Wednesday song should feel like the track playing at The Bronze/P3 equivalent while the episode's lesson lands.

Internal note: DJ JAIDY is Josh Hudson.

Feature idea: let members put in a song request by entering inspiration and selecting a 90s/Y2K song style.

Song prompt ideas:

- an ode to Deb, who is terrible at her job but takes no shit from anyone
- a different smart insult for every day of the week based on a terrible coworker everyone has

Suggested per-song fields:

- issue number
- song title
- audio link or embed
- short description
- club-scene liner note
- episode tie-in
- requested by, if member-submitted
- inspiration text
- selected style
- publish status
