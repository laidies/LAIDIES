# lAIdies Lesson Printable Takeaway Template

Use this structure for each Wednesday lesson so subscribers get something more durable than the games: a one-page desk reference they can print, annotate, and reuse.

## Recommended Format

**Standard Footer**
Add this to every page of every printable:

`© 2026 lAIdies™ / Ali Eakin. Personal-use subscriber printable. Do not resell, repackage, redistribute, or use in paid training without permission. lAIdies.com`

**Title**
Name the skill in plain language.

Example: `Prompt Like You Mean It`, `Receipts Before Reliance`, `Make AI Ask Better Questions`

**One-Line Lesson**
The practical idea in one sentence.

Example: `AI does better work when your prompt sounds like a clear brief, not a vague wish.`

**Why This Matters**
Connect the skill to a busy corporate woman's real day: meetings, emails, deadlines, stakeholders, confidence, time.

**The Framework**
Give the repeatable system. Keep it short enough to remember.

Example:
- `BRIEF`: Background, Role + Reader, Instructions, Examples + Edges, Format
- `TRY-ON`: Task, Reader, Yardstick, Output, Next revision
- `RECEIPTS`: Reference, Evidence, Constraints, Exceptions, Inspect, Trust but verify, Sensitive data

**Copy-Paste Prompt**
Give one strong starter prompt with blanks.

**Before / After**
Show a weak prompt or workflow, then the upgraded version.

**Troubleshooting**
List 3 to 5 common failures and the fix.

Example:
- Too generic: add audience, stakes, and a good example.
- Too long: set a word count and priority order.
- Invented details: require source-only answers and missing-info labels.

**Analogy**
Anchor the lesson in the lAIdies reference world.

Example:
- Clueless closet: AI can pull options, but you still style the final outfit.
- David Rose specificity: vague taste produces chaotic results.
- The fitting room: the first answer is a try-on.

**The Thursday Try-On**
Give one 5-minute homework task that uses the skill on real work.

Example:
`Take one email you need to send this week. Ask AI for three versions: warmer, shorter, and more executive. Pick one line from each.`

**Receipts / Safety Note**
Add a small box for anything that should be verified, kept private, or treated carefully.

**Prompt Closet**
Give subscribers one line to save:
`Save this when you need: [use case].`

## Visual Pattern

Each printable should feel like a polished desk insert from the lAIdies reference closet:
- Hot pink accent, dark ink, one secondary accent color.
- One generated image or graphic per printable.
- A small "receipt" or checklist box.
- One playful analogy, but the core content stays useful.
- Enough white space for notes.
- lAIdies ownership and personal-use language on every page, small but visible.

## Subscriber-Gated Rule

Use this default access pattern:

- Public page: title, preview image/contact sheet, short description, and one sample section.
- Subscriber gate: full printable download.
- Sharing rule: people can share the public page link, not upload or resell the file.
- Paid/member version later: deeper workbook, facilitator notes, templates, and lesson bundle.

## Suggested Printable Series

1. Prompt Like You Mean It - prompting basics.
2. Receipts Before Reliance - verification, sources, and hallucination control.
3. Make AI Ask Better Questions - clarifying questions and missing context.
4. First Draft Fitting Room - iteration and versioning.
5. The Meeting Prep Machine - agendas, objections, scripts, follow-ups.
6. Rewrite Without Losing Yourself - tone, voice, and editing.
7. The Reference Closet Method - using examples and source material.
8. From Notes to Next Steps - summarization into action.
9. The Smart Skeptic - pressure-testing ideas.
10. Build a Personal Prompt Closet - saving reusable workflows.
