# Prompt Cheat Sheet Source Note

This source note supports `prompt-cheat-sheet.html`. It is not meant to be printed every time, but it gives the reasoning and reference trail behind the lAIdies prompting guide.

## High-Signal Best Practices Used

- Be clear and specific about the task, audience, desired output, and constraints.
- Provide background context and source material instead of expecting the model to infer everything.
- Use examples when taste, tone, structure, or quality matter.
- Break complex tasks into steps or staged workflows.
- Ask for the output format explicitly.
- Iterate. Treat the first answer as a draft or try-on.
- For important facts, current details, sensitive decisions, or regulated topics, verify with reliable sources and do not rely on AI alone.
- Ask the model to identify assumptions, missing context, uncertainty, and risks.
