# Website Feature Social Plan

Use this to add the fun website pieces into Instagram, instead of only posting issue summaries.

The goal is simple: social should make people want to click, play, answer, and come back.

## Feature Names To Use

Use the names exactly like this:

- Madame CLAI-O
- Ask LAIDY
- Dream Phone
- Girl Talk Prompt Deck
- Five-Minute Try-On
- Trading Card Pack
- Issue Quiz
- Glossary Rolodex
- Reference Closet
- Hot Goss
- House DJ
- The lAIdies Room
- Try-On Debrief
- Ask the Room
- Burn Book
- Member Cards

Important: use `Madame CLAI-O`, not `Madame Cleo`. The bit should feel like a broad 90s psychic-hotline homage, not a direct copy of Miss Cleo's name, likeness, voice, accent, or biography.

## How To Work These Into Issue 01 And Issue 02

Do not replace the issue posts. Add these as click/play/community posts around them.

For each issue, use:

- 3 issue posts: Reel, carousel, Stories.
- 2 feature posts: one website feature Reel/static post, one Story sequence.
- 1 community prompt: Ask the Room, Try-On Debrief, Burn Book, or Member Cards.

That makes the website feel alive without making Ali post every single day forever.

## Recommended Instagram Rhythm

Use this pattern for the first two issues:

- Monday: issue Reel
- Tuesday: feature Reel or static post
- Wednesday: issue carousel
- Thursday: feature Stories with poll/question/link
- Friday: community prompt

If that is too much, use:

- one issue Reel
- one issue carousel
- one feature Story sequence
- one community question

## Posting Notes

Feature posts should always do one of these:

- send people to a specific page
- ask a question
- make people play a tiny game
- invite a community reply
- connect back to the current issue

Do not post generic "check out the website" captions. Every post should point to one specific thing.
