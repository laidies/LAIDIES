# lAIdies Website Link Map For Social

Use this when adding Story link stickers, caption CTAs, Canva notes, or scheduling links.

Replace `[site]` with the live website root once deployed.

## Platform Rule

Instagram Story link stickers can link directly to a specific page or section.

Instagram feed images cannot contain clickable links after posting. For feed posts, use:

- a clear caption CTA
- the link in bio
- a Story reshare with a link sticker
- a QR code only when the visual needs to work off-platform too

LinkedIn post links are clickable, so use the direct destination link.
