## Before You Start

Use desktop LinkedIn if possible. LinkedIn says Page creation is supported on desktop and iOS, but not Android mobile.

You will create a Company Page, not a Showcase Page.

## Step-By-Step

1. Go to LinkedIn.
2. Click `For Business` in the top-right navigation.
3. Scroll down.
4. Click `Create a Company Page`.
5. Choose `Company`.
6. Fill out the Page fields using the copy below.
7. Check the verification box confirming you can create the Page.
8. Click `Create page`.
9. Click `Start building your page`.
10. Upload the logo and cover image.
11. Add the overview/about copy.
12. Add the website URL once the live URL is ready.
13. Publish.
