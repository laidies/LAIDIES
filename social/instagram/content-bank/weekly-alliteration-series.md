# Weekly Alliteration Series

Recurring weekday social formats for `@we.are.laidies`.

The core weekly spine should stay light:

- Monday: prompt/content setup
- Tuesday: issue teaser
- Wednesday: issue drop
- Thursday: practice the lesson
- Friday/weekend: optional community, news, playlist, or small-sip follow-up

## Recommended Weekly Set

### Makeover Monday

Use for prompt glow-ups, messy drafts, and Ask LAIDY-style rewrites.

```text
Makeover Monday:
bad prompt in, better brief out.
```

Reference energy:

```text
Clueless closet makeover, David Rose specificity, America's Next Top Model feedback.
```

### Terry Tuesday

Use for the White Chicks / "A Thousand Miles" Tuesday teaser.

```text
statement [clip]
statement [clip]
statement [clip]
new issue drops Wednesday
```

Use the full pack:

```text
social/instagram/content-bank/terry-tuesday.md
```

### On Wednesdays We Use AI

Use for the actual issue drop.

```text
New lAIdies issue is live.
Read it, take the quiz, pull the card, try the prompt.
```

Reference energy:

```text
Mean Girls, the standing lAIdies line, weekly ritual.
```

### The Thursday Try-On

Use the day after the issue drops. This is where readers practice the Wednesday lesson.

Core structure:

```text
Yesterday we learned [concept].
Today, try it on.
```

Issue 1 example:

```text
Yesterday we opened the tab.
Today, give ChatGPT, Claude, and Gemini the same low-risk prompt.
```

Issue 2 example:

```text
Yesterday we learned specificity.
Today, ask AI once vaguely, then ask again with audience, context, tone, length, and what not to include.
```

CTA:

```text
Bring the messy version to the Try-On Debrief.
```

Reference energy:

```text
Fitting room, Clueless closet computer, homework that does not feel like homework.
```


Reference energy:


### Small Sip Sunday

Use for one tiny low-pressure AI action before the week starts.

```text
Small Sip Sunday:
one tiny AI move for the week ahead.
```

Reference energy:

```text
Small sips, big moves.
```
